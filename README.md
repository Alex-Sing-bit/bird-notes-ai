# Bird Notes AI — RAG-система для идентификации птиц по текстовым заметкам

**Bird Notes AI** — это веб-приложение, которое позволяет загрузить текстовую заметку о птице и получить умный ответ о том, какие виды птиц были описаны. Система использует семантический поиск (RAG) на основе эмбеддингов и генеративную модель Gemini от Google.

---

Ключевые компоненты:

| Компонент | Назначение |
|-----------|------------|
| Flask Backend | Обработка запросов, аутентификация, управление сессиями |
| ChromaDB | Хранение и семантический поиск эмбеддингов статей о птицах |
| Embedding Model | Преобразование текста в векторное представление |
| Gemini API | Генерация ответа на основе найденного контекста |
| SQLite | Хранение пользователей, истории запросов, метаданных |

## Как это работает

### 1. Наполнение базы знаний (ETL)

1. Статьи о птицах извлекаются через Wikipedia API.
2. Текст очищается и разбивается на смысловые чанки.
3. Для каждого чанка генерируется эмбеддинг с помощью модели intfloat/multilingual-e5-small.
4. Векторы сохраняются в ChromaDB вместе с метаданными.

### 2. Пользовательский запрос

1. Пользователь вводит текстовую заметку (например, "Видел крупную черную птицу с оранжевым клювом").
2. Заметка преобразуется в эмбеддинг.
3. Система ищет топ-10 похожих чанков в ChromaDB.

### 3. Генерация ответа

1. Найденные чанки подставляются в промпт для Gemini API.
2. Модель анализирует контекст и возвращает ответ: какие птицы вероятно были увидены.

## Установка и запуск

### Требования

- Python 3.10+
- Google Cloud проект с включенным OAuth 2.0
- API-ключ Google Gemini

### Локальный запуск

1. Клонируйте репозиторий:

```bash
git clone https://github.com/Alex-Sing-bit/bird-notes-ai.git
cd bird-notes-ai
```

2. Создайте виртуальное окружение:

```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```

4. Настройте конфигурацию:

```bash
cp config_ex.py config.py
```

Отредактируйте `config.py`:
- Вставьте `GOOGLE_CLIENT_ID` и `GOOGLE_CLIENT_SECRET`
- Вставьте `GEMINI_API_KEY`
- При необходимости измените `SECRET_KEY`

5. Запустите приложение:

```bash
python app.py
```

6. Откройте браузер по адресу: `http://localhost:5000`
